# FinSight AI — Finance Analyser

> AI-powered personal finance analyser with LLM-driven budget insights and spending categorisation

![Status](https://img.shields.io/badge/status-In-Progress-blue)
![Month](https://img.shields.io/badge/roadmap-Month-2-purple)

## 📋 Overview

AI-powered personal finance analyser with LLM-driven budget insights and spending categorisation. Part of a 5-month Cybersecurity + AI + Automation roadmap targeting $2K MRR.

## ✨ Features

- CSV / bank statement upload and parsing
- AI spending categorisation
- Budget recommendations engine
- Exportable PDF/Markdown report

## 🛠️ Tech Stack

- Python 3.11+
- FastAPI
- OpenAI / Claude API
- Pandas
- Streamlit

## 🚀 Getting Started

```bash
git clone https://github.com/malikatemz/finsight-ai.git
cd finsight-ai
pip install -r requirements.txt
python main.py
```
